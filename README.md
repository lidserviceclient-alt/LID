# LID Backoffice

Interface backoffice e-commerce (React + Vite + Tailwind).

## Lancer en local

```bash
npm install
npm run dev
```

## API / Auth (local)

- URL API : `VITE_BACKOFFICE_API_URL` (par dĂŠfaut `http://localhost:9000` via `.env.local`)
- Compte admin (seed `local`) : `admin@demo.com` / `Admin123!`

## Structure

- `src/components` : UI et layout
- `src/pages` : pages metier
- `src/data` : donnees mock
- `src/styles` : styles globaux
